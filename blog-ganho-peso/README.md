# Curva Saudável - Blog de Ganho de Peso para Magros

Blog completo com receitas caseiras e dicas de autoestima para pessoas magras que querem ganhar peso de forma saudável.

## Funcionalidades

- **6 Receitas caseiras** detalhadas com ingredientes e preparo
- **Dicas de autoestima** para motivar os leitores
- **Rotina diária** sugerida para ganho de peso
- **Calculadora de calorias** interativa
- **Mensagens motivacionais** aleatórias
- **Design responsivo** para celular e desktop
- **Animações suaves** e transições

## Receitas Incluídas

1. Vitamina de Banana com Amendoim (~450 kcal)
2. Omelete Recheada com Queijo (~380 kcal)
3. Paçoca de Amendoim Caseira (~520 kcal)
4. Abacate com Ovo e Pão (~480 kcal)
5. Macarrão com Molho Cremoso (~650 kcal)
6. Shake de Ganho de Peso (~550 kcal)

## Como Usar

1. Abra o arquivo `index.html` no navegador
2. Navegue pelas seções
3. Clique em "Ver Receitas" para ver todas as opções
4. Use a calculadora de calorias (aparece após 10 segundos)

## Deploy na Vercel/Netlify

### Vercel
1. Crie uma conta no [Vercel](https://vercel.com)
2. Conecte seu repositório Git
3. O deploy será automático

### Netlify
1. Crie uma conta no [Netlify](https://netlify.com)
2. Arraste a pasta do projeto ou conecte o Git
3. Configure como site estático

## Personalização

Para adicionar novas receitas:
1. Copie um dos cards existentes em `index.html`
2. Modifique o conteúdo
3. Adicione na grid de receitas
